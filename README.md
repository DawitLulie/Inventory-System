# Java Inventory System

## Description
A simple console-based **Java Inventory System** that helps manage items in stock.  
Users can **add, remove, display, and track items** with their name, quantity, and price.

## Features
- Add new items with **name, quantity, and price**  
- Remove items by **name**  
- Display all items in inventory  
- Track **total number of items**  
- Simple **menu-driven interface** for easy interaction  


## How to Run
1. Ensure **Java JDK** is installed (version 8 or higher)  
2. Open a terminal in the project folder  
3. Compile the Java files:
```bash
javac Item.java Main.java
